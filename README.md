# Sistema Acadêmico AGIS

Sistema de gestão acadêmica desenvolvido para a disciplina de Laboratório de Banco de Dados, focado no gerenciamento de alunos, cursos e matrículas. O projeto aplica conceitos de **Arquitetura MVC**, persistência de dados com **JDBC** e regras de negócio integradas ao **SQL Server** com utilização de procedures.

## Tecnologias Utilizadas

- **Java 21** com Jakarta EE (Servlets & JSP)
- **Apache Tomcat 10.1.54
- **Microsoft SQL Server**
- **JSTL** 
- **Bootstrap 5** para interface responsiva

---

## Arquitetura do Projeto

O sistema foi estruturado seguindo o padrão **MVC (Model-View-Controller)** para garantir a separação de responsabilidades.
